## Project Overview
This repository contains the source code for the academic portfolio of Dr. Arch. Batel Yossef Ravid.

### Key Features
- **Dynamic 3D Background:** Three.js visualization of urban data (Grayscale theme).
- **Interactive Maps:** Leaflet integration for project mapping.
- **Responsive Design:** Optimized for Desktop, Tablet, and Mobile.
- **Teaching & Research Sections:** Comprehensive documentation of academic work.
